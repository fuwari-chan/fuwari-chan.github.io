Compile MYSQL On Redhat#

Apa Itu MySQL ?#

¬† ¬† MySQL (baca: MY-ES-KYOO-EL) adalah perangkat lunak manajemen basis data relasional yang menggunakan bahasa SQL. Sistem ini didesain agar data dapat diakses, diatur, dan dimanipulasi secara efisien. MySQL berfungsi untuk memastikan data tersimpan secara akurat dan dapat diambil kapan saja.

¬† ¬† Adapun SQL dan MySQL memiliki keterkaitan erat tetapi berbeda dalam fungsi. SQL adalah bahasa pemrograman digunakan untuk mengelola dan memanipulasi data dalam basis data. Sementara itu, MySQL adalah perangkat lunak yang menggunakan SQL sebagai cara untuk berinteraksi dengan data dalam basis datanya.

Catatan

Pengertian singkat karena yang akan kita bahas ,bagaimana cara compile mysql di linux arm64/android

Apa itu Compile ?#

¬† ¬† Pengertian compile adalah proses pengubahan kode program (Source code) menjadi kode objek atau sebuah file, menggunakan aplikasi yang disebut Compiler.

1. Menginstall DevTool#

¬† ¬† Sebelum kita mengcompile sebuah program tentunya kita persiapkan alat yang dibutuhkan,program yang kita buat menetukan alat apa yang kita gunakan,dan proses nya pun akan berbeda-beda disesuaikan dengan program yang akan kita buat,
¬† ¬† Contoh : Kita akan membuat mie instan tentu harus dipersiapkan alat-alat untuk mememasak nya dan ada tahap proses memasaknya,pasti akan berbeda dengan membuat nasi goreng ,sate dan lain-lain!

¬† ¬† Untuk menginstall devtool di redhat arm64 mungkin nama package akan berbeda di linux lain nya namun tetap memiliki fungsi yang sama !
¬† ¬† Sebelum menginstall devtool alangkah baik nya kalian sudah menginstall repo eternal untuk redhat!

  • Make DIR and Permission
### for socket
/var/run/mysql/mysql.sock

sudo mkdir /var/run/mysql

sudo chmod -R 777 /var/run/mysql/

### for datadir
/www/server/data

sudo mkdir -p /www/server/data

sudo chmod -R 777 /www/server/data

Extract MySQL#

mv Download/mysql-5.7.38.zip /opt/

or

mv /sdcard/mysql-5.7.38.zip /opt/

cd /opt/

unzip mysql-5.7.38.zip

cd mysql-5.7.38

Download Boost#

cmake . -DDOWNLOAD_BOOST=1 -DWITH_BOOST=boost

Configure Manual (no recomendation)#

cmake \
-DWITH_BOOST=/usr/local/src/mysql-5.7.38/boost/boost_1_59_0 \
-DCMAKE_INSTALL_PREFIX=/usr/local/mysql \
-DWITH_INNOBASE_STORAGE_ENGINE=1\
-DDEFAULT_CHARSET=utf8 \
-DDEFAULT_COLLATION=utf8_general_ci \
-DMYSQL_UNIX_ADDR=/tmp/mysql.sock \
-DMYSQL_DATADIR=/usr/local/mysql/data \
-DSYSCONFDIR=/usr/local/mysql/etc \

Configure Auto#

don't forget check your path boost!!!

cmake .

### Make it

make

### Compile and Install

sudo make install
