Basic Custom Menu XFCE4#

  • Apa itu Xfce4 ?

¬† ¬† Xfce adalah desktop environtmen ringan untuk sistem operasi seperti UNIX. Tujuan utamanya adalah cepat dan rendah sumber daya sistem, namun tetap menarik secara visual dan mudah digunakan.

Catatan

Hanya Pengertian singkat saja ,karena kita akan membahas kepada inti customisasi xfce4

Persyaratan#

  1. System Operasi Linux Sudah Terinstall XFCE
  2. Text Editor yang mudah di gunakan

Struktur Folder Penting#

[root]-/etc/
  	|
        xdg/menus/
        |
        xfce-applications.menu
      --usr/
  	|
      --share/application/
	     |
          list-file of
             .desktop
      --share/desktop-directories/
	     |
           list-file of
             .directory

Penjelasan#

[Desktop Entry] beri deskripsi

Version=1.0 versi

Type beri nilai Application

Exec beri nilai tergantung aplikasi apa/script/program sebagai contoh /opt/edex-ui-arm64/edex-ui --no-sandbox

Icon nilai /opt/edex-ui-arm64/edex-ui.png source/resource image/icon untuk applikasinya bukan menu,sebmenu,atau deepermenu,untuk menu,submenu dan deepermenu resource akan terbaca di file direcktory di source /usr/share/desktop-directories

StartupNotify notification aplikasi false atau true

Terminal jika program atau aplikasi gui tanpa debug bisa bseri nilai false jika aplikasi/program itu berjalan di terminal atau application CLI maka nilai harus true

Categories di beri nilai Desktop-Edex; agar aplikasi akan terentri di menu,submenu atau di deepermenu ,contoh Desktop-Edex adalah submenu,akan terbaca oleh xfce4-applications.menu di source /etc/xdg/menus

OnlyShowIn di beri nilai XFCE; jadi akan terbaca di desktop environtmen xfce4 saja !

Name adalah nama untuk app yang akan di list menu ,sebagai contoh Redhat-Edex-UI
